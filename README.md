esp-gti ⚡

ESP-based grid-tie inverter controller for Home Assistant.

This project was created to dynamically control the output power of a grid-tie inverter using an ESP32-C5 and an MCP41 digital potentiometer.
The goal is simple: smarter and more flexible solar power control directly integrated into Home Assistant via ESPHome.

Built as an open-source DIY project for makers, tinkerers and Home Assistant enthusiasts. ☀️

✨ Features

* Dynamic inverter power control
* ESPHome integration
* Home Assistant support
* MCP41 digital potentiometer control
* Real-time monitoring
* Open-source DIY approach

⸻

🛠 Hardware

Main Components

* ESP32-C5
* MCP41-503 digital potentiometer
* Grid-tie inverter

⸻

🖥 Integration

The controller integrates directly with:

* ESPHome
* Home Assistant
* MQTT (optional)

Possible use cases include:

* Dynamic inverter regulation
* Battery-aware power control
* Smart solar automation
* Energy balancing
